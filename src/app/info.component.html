What is Data Commandr

Data Commandr is a self-service tool for mass market which is positioned as an "Excel for Big Data" or "Big Data Calculator". It was created with a single mission: radically simplifying and democratizing all kinds of operations with data for everyone with little or no help from IT. By making data manipulations easier for non-IT users, Data Commandr dramatically shortens the total time for data integration, transformation and analysis. Data Commandr is aimed at creating new data mash-ups in seconds by integrating numerous heterogeneous data sources. It revolutionizes the way data integration is done by combining data transformation and analysis. Data Commandr employs the familiar spreadsheet concept but instead of cell-formulas it uses column-formulas.

Self-service and user-driven

No experience in mathematics or statistics is needed. Assistance engine and automatic recommendations at all stages based on data semantics, automatically discovered relationships, user behavior analysis and the current context.

Agile and ad-hoc

Play with data in an agile manner by changing transformation and computation options. Define arbitrary custom transformations and computations without predefined scenarios by using an easy to use but powerful analytical expression language.

Real time and responsive

Instant reaction even for large data sets thanks to a novel column-oriented in-memory data processing engine.

For whom Data Commandr is Useful (Customers)

Data Commandr is intended for a broad range of non-IT users starting from data enthusiasts, casual users or business users, and ending with data artisans and analysts. All these users do not possess deep knowledge in mathematics and statistics but need some simple yet powerful tool to solve a problem or answer a question by analyzing available data. The buyers of the product are companies which want to reduce their IT costs by getting analytical tools suitable for business users.

What Problems Data Commandr Solves (Pains)

A typical problem for businesses is that there exist many corporate and external data sources but none of them has data the user needs. This problem is getting even more acute with the explosion of data volume and the variety of data sources - the Big Data problem. In this case, data integration, transformation and other "data wrangling" tasks account for a great deal or even most of the difficulties in data analysis and constitute the most tedious part of the overall analysis process. Such data wrangling tasks are normally performed by experts from the IT department by using classical BI tools. This traditional approach is not only expensive and slow but also error-prone because frequently IT department do not understand the needs of domain-experts.

How Data Commandr Works and Competitive Advantages (USP)

Novel technology

Data Commandr is based on a unique disruptive technology for data management and analytics collectively referred to as concept-orientation. It originates from almost a decade of research performed by Dr. Alexandr Savinov at several research organizations and published in more than 20 academic papers: http://conceptoriented.org.

New design

Data Commandr employs a novel design of the product. It is based on the assumption that the user can define new data columns based on the existing data columns similar to how cells in spreadsheet applications are defined in terms of other cells. In this sense, Data Commandr could be characterized as Excel for column-based analytical data integration and transformation.

Value Proposition Benefits (Gains)

Ease of use

Data Commandr makes complex analytical data integration possible for non-IT users.

Fast time-to-value

Data Commandr dramatically shortens the total time for data integration, transformation and analysis from months to minutes.

Lower TCO

Data Commandr transfers the burden of authoring BI contents to the end users who know much better what needs to be done.
